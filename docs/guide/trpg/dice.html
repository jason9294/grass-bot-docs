# 骰子功能

便於TRPG遊戲中常見幾種骰子功能。

# 使用方式

# xdy 骰子

於聊天頻道中以<xdy>格式輸入，即可使用基本骰子指令，x代表骰子的顆數，y代表骰子的面數。

# 範例

輸入: 1d10 //代表骰一顆十面骰
輸入: 2d100 //代表骰二顆百面骰

支援四則運算式

1d10+1

1d10+1*2

(1d10+1)/4

注意

括號內先計算，先乘除後加減。
